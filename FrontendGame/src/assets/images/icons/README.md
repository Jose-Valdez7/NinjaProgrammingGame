# Iconos

Esta carpeta contiene iconos y elementos gráficos pequeños:

- `sword.svg` - Icono de espada
- `shield.svg` - Icono de escudo
- `coin.svg` - Icono de moneda
- `heart.svg` - Icono de vida
- `star.svg` - Icono de estrella/puntos

**Formatos recomendados:** SVG para iconos escalables, PNG para iconos con efectos
**Tamaño sugerido:** 24x24px, 32x32px o 48x48px
