# Interfaz de Usuario

Esta carpeta contiene elementos de la interfaz de usuario:

- `button.png` - Botones del juego
- `panel.png` - Paneles de información
- `progress-bar.png` - Barras de progreso
- `logo.png` - Logo del juego
- `menu-items.png` - Elementos del menú

**Formatos recomendados:** PNG con transparencia
**Tamaño sugerido:** Según el diseño de tu UI
