# Personajes

Esta carpeta contiene las imágenes de los personajes del juego:

- `ninja.png` - Imagen del personaje principal (ninja)
- `enemy.png` - Imágenes de enemigos
- `npc.png` - Personajes no jugables
- `boss.png` - Jefes de nivel

**Formatos recomendados:** PNG con transparencia
**Tamaño sugerido:** 64x64px o 128x128px para sprites
